**DECOFORMATIC** es una herramienta local de escritorio interactiva diseñada en Python para la codificación y descodificación de texto a flujo binario en tiempo real. Con una estética retro-terminal inspirada en entornos cyberpunk, cuenta con una economía digital integrada basada en **DECOTOKENS** y un sistema de automatización de créditos.

---

## ✨ Características Principales

* 🧠 **Traducción en Tiempo Real:** Convierte texto plano a bloques binarios de 8 bits (y viceversa) al instante a través de una consola optimizada.
* 🪙 **Wallet de DECOTOKENS Local:** Gestión de transacciones local sin necesidad de bases de datos externas mediante un archivo de configuración `config.json` encriptado en la ruta raíz.
* ⚡ **Auto-Refresh de Créditos:** Olvídate de canjear créditos a mano. Cuando el buffer de operaciones llega a 0, el núcleo de la aplicación debita automáticamente 1 DECOTOKEN para inyectar +20 usos de forma invisible.
* 🎁 **Suscripción Mensual Gratuita:** El sistema monitoriza el tiempo real y realiza un ingreso automático de **1500 DECOTOKENS cada mes** al iniciar la app.
* 👾 **Easter Egg Integrado:** Consola de inyección de código secreta para desarrolladores y auditorías.

---

## 📊 Especificaciones del Sistema de Créditos

| Unidad | Equivalencia | Automatización |
| :--- | :--- | :--- |
| **1 DECOTOKEN** | +20 Codificaciones / +20 Descodificaciones | Consumo automático al agotar el buffer |
| **Subscripción** | +1500 DECOTOKENS / Mes | Inyección automática al detectar cambio de mes |
